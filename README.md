# Memory-game

Цей проект відповідає наступним вимогам:

1. Використання подій для взаємодії з користувачем: Я використовую події кліку для початку гри та перевертання карток.
2. Динамічна зміна контенту DOM: Картки створюються та оновлюються динамічно за допомогою JavaScript.
3. Використання ООП: Весь функціонал гри інкапсульований у клас MemoryGame.
4. Використання асинхронного коду: Я використовую setTimeout для затримки перевірки збігу карток.
5. Використання сховищ даних: Я використовую localStorage для збереження найкращих результатів.
6. Використання функцій із ES6: Я використовую стрілочні функції, деструктуризацію та інші можливості ES6.


# Додаткові особливості:

- Гра має таймер та систему балів.
- Картки перемішуються при кожному запуску гри.
- Реалізована анімація перевертання карток за допомогою CSS.
- Можливість паузи гри.
- Додано responsive.

# Додано рівні складності:

1. Новий елемент вибору рівня в HTML.
2. Метод setLevel() для налаштування параметрів гри відповідно до обраного рівня.
3. Різна кількість часу та карток для кожного рівня.

# Реалізовано логіку створення та відображення карток:

1. Динамічна зміна розміру сітки карток залежно від їх кількості.
2. Вибір підмножини символів залежно від рівня складності.
3. Реалізовано логіку збереження рекордів: зберігається рахунок та рівень складності.
4. Додано блокування елементів керування під час гри:
5. Кнопка "Почати гру" та вибір рівня блокуються під час активної гри.
6. Кнопка "Пауза" активується тільки під час гри.
